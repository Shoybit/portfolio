# 🌐 Personal Portfolio Website

A modern, responsive, and animated **developer portfolio** built with
**Next.js, React, Tailwind CSS, and Framer Motion**.

------------------------------------------------------------------------

## ✨ Features

-   ⚡ Next.js 16 + React 19\
-   🎨 Tailwind CSS 4\
-   🎞️ Framer Motion animations\
-   📩 EmailJS contact form\
-   📱 Fully responsive\
-   🔥 Clean UI

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   Next.js\
-   React\
-   Tailwind CSS\
-   Framer Motion\
-   FontAwesome\
-   EmailJS

------------------------------------------------------------------------

## 📦 Installation

``` bash
git clone https://github.com/Shoybit/portfolio.git
cd portfolio
npm install
npm run dev
```

------------------------------------------------------------------------

## 📩 EmailJS Setup

1.  Create an account at https://emailjs.com\
2.  Create Service ID, Template ID & Public Key\
3.  Replace values in sendForm()

``` ts
emailjs.sendForm(
  "YOUR_SERVICE_ID",
  "YOUR_TEMPLATE_ID",
  e.currentTarget,
  "YOUR_PUBLIC_KEY"
);
```
